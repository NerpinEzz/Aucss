# information 
>aucss is a css framework where the use of css can be directly implemented into html via class name


>how to install aucss 

```
git clone https://github.com/NerpinEzz/Aucss.git
```
>furthermore 

```
 <link rel="stylesheet" href="/cssin/cssin.css">
```

>example 
```
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=Edge">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <title>HTML</title>
  
  <!-- HTML -->
  

  <!-- Custom Styles -->
  <link rel="stylesheet" href="cssin.css">
</head>

<body>
<!--navbar -->
<div class="w100 h50 bg-sky ">
   <div class="flex jc-center ai c-white lh-55">Navbar
   </div>
</div>
<div class="w100 h50 bg-black flex">
   <div class="c-white lh-55 pe-l10">Navbar
   </div>
   <div class="lh-55 mr-r45 c-white-a">
       <ul class="flex ">
           <li class="mr-r5"><a href="">home</a></li>
           <li class="mr-r5"><a href="">kontak</a></li>
           
       </ul>
   </div>
</div>

<!--card-->
<div class="card w25 bg-white">
    <div class="card-img">
        <img src="foto.jpg" alt="">

</div>
  <!-- Project -->

</html>
```
